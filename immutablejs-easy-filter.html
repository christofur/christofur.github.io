Easy filter in Immutable.js

Posted on Wed 13 January 2016 in JavaScript

Here's a teeny snippet of a super easy filter method using Immutable.js

//Create a map from some JavaScript-like input
let input = Immutable.fromJS({a:1,b:2,c:3,d:4});

//Filter the output, returning only items where the predicate is true.
let output = input.filter(x => x % 2 === 0); // { "b": 2, "d": 4 }

Enjoy playing!
